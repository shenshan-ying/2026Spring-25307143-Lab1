# 桌面宠物球应用

## 项目简介

桌面宠物球是一款运行在HarmonyOS系统上的桌面宠物应用，支持多种宠物球类型、宠物培养系统、多设备间传递等功能。

## 开发环境要求

- DevEco Studio 3.1 或更高版本
- HarmonyOS SDK API Version 9 或更高版本
- Node.js 14.x 或更高版本

## 项目配置说明

### 1. 项目基本信息

- **应用包名**: `com.example.desktoppetball`
- **应用版本**: 1.0.0
- **编译SDK版本**: API 9
- **支持设备**: 手机、平板

### 2. 权限配置

应用需要以下权限：

- `ohos.permission.SYSTEM_FLOAT_WINDOW`: 悬浮窗权限（用于桌面显示宠物球）
- `ohos.permission.KEEP_BACKGROUND_RUNNING`: 后台运行权限（保持宠物球持续运行）
- `ohos.permission.CAMERA`: 相机权限（拍摄自定义宠物球）
- `ohos.permission.READ_MEDIA`: 相册权限（选择自定义宠物球图片）

### 3. 在DevEco Studio中打开项目

#### 步骤1：打开项目

1. 启动 DevEco Studio
2. 选择 `File` -> `Open`
3. 选择项目根目录：`D:\ClassWork\Desktop pet ball`
4. 等待项目加载完成

#### 步骤2：配置签名

1. 选择 `File` -> `Project Structure` -> `Project` -> `Signing Configs`
2. 点击 `Automatically generate signature`
3. 登录华为开发者账号
4. 等待签名自动生成完成

#### 步骤3：同步项目

1. 点击工具栏的 `Sync Now` 按钮
2. 等待Gradle同步完成
3. 如果出现依赖下载失败，请检查网络连接或配置镜像源

### 4. 运行和调试

#### 在模拟器上运行

1. 选择 `Tools` -> `Device Manager`
2. 创建或启动一个HarmonyOS模拟器
3. 点击工具栏的运行按钮（绿色三角形）
4. 选择目标设备并运行

#### 在真机上运行

1. 连接HarmonyOS设备到电脑
2. 确保设备已开启开发者模式和USB调试
3. 点击工具栏的运行按钮
4. 选择连接的设备并运行

#### 调试应用

1. 在代码中设置断点
2. 点击工具栏的调试按钮（绿色虫子图标）
3. 选择目标设备并开始调试
4. 使用调试工具查看变量、调用栈等信息

### 5. 常见问题解决

#### 问题1：Gradle同步失败

**解决方案**：
- 检查网络连接
- 配置国内镜像源
- 清理项目缓存：`Build` -> `Clean Project`

#### 问题2：签名配置失败

**解决方案**：
- 确保已注册华为开发者账号
- 检查网络连接
- 手动配置签名证书

#### 问题3：模拟器启动失败

**解决方案**：
- 检查电脑是否支持虚拟化
- 更新DevEco Studio到最新版本
- 重新创建模拟器

#### 问题4：权限申请失败

**解决方案**：
- 检查module.json5中的权限配置
- 确保权限reason字符串已定义
- 在设备设置中手动授予权限

│   ├── state/             # 状态管理

## 核心功能

1. **桌面悬浮显示**：宠物球在桌面悬浮显示
2. **运动模式**：自由滚动、静止模式
3. **弹弓发射**：拖动到边缘可弹射
4. **宠物培养**：好感值积累、状态转换
5. **碰撞系统**：多宠物球碰撞检测
6. **资源管理**：预设和自定义宠物球
7. **分布式能力**：跨设备传递宠物球

## 技术架构

采用四层架构设计：

1. **基础设施层**：数据访问、工具组件、API封装
2. **服务层**：系统服务适配、应用服务
3. **业务逻辑层**：核心业务引擎、管理器
4. **表示层**：UI组件、动画、状态管理
